# 硬體接線與邏輯通道對照（ESP32-C3 測試板）

## 邏輯通道 0–8

與韌體 `Config.h` 中 `ADC_PINS`、`DIGITAL_PINS` 及 Web `web/core/state.js` 之 `PINS_CONFIG` 一致。

| 邏輯 ID | GPIO | 預設用途建議 |
|---------|------|----------------|
| 0–4 | 0, 1, 2, 3, 4 | **ADC1** 類比實驗（GPIO2 為支援／複合腳位） |
| 5–6 | 8, 9 | 數位輸入 |
| 7–8 | 20, 21 | 數位輸入；韌體預設常開啟內部上拉（`pullupMask`） |

## 實驗建議

- **電子琴／多鍵**：優先使用**已啟用**且乾淨的類比或數位通道；觸控模式可同一腳多鍵並行（見協定 `touchModeMask`）。
- **弓箭／張力**：多使用 ADC 0–4；依實際線長調整取樣率與濾波（僅網頁端）。
- **共地**：外部感測器與板子需**共地**；懸空 ADC 易雜訊，教學上可接固定參考或使用軟體中位數／濾波。

## 素材檔案

實驗專屬 STL、組裝圖、照片請放在各實驗資料夾下之 `hardware/`（例如 `experiments/omni-forge-bowsmith/hardware/`）。
